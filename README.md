# simple_diagram_dashboard_code# Simple Diagram Dashboard

A React-based dashboard application for uploading and viewing circuit diagrams with component listings.

## Features

- **Image Upload** - Upload PNG/JPG images via drag & drop or file browser
- **Diagram Viewer** - Zoom, pan, and reset view functionality
- **Components List** - Display circuit components with mock API data
- **Component Selection** - Click components to highlight them
**Responsive Design** - Works on desktop, tablet, and mobile

## Technologies Used

- React 18 (Functional Components, Hooks)
- React Dropzone for file uploads
- React Zoom Pan Pinch for image controls
- Axios for API calls (mock implementation)
- Lucide React for icons
- CSS3 with Flexbox/Grid for layouts
